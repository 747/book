# 列物第六

```
吾有七列。名之曰「其所」。曰「所難」。曰「所沾」。曰「所贈」。曰「所報」。曰「所感」。曰「所傷」。
充「其所」以「「太山」」以「「桂林」」以「「漢陽」」以「「雁門」」。
充「所難」以「「樑父艱」」以「「湘水深」」以「「隴阪長」」以「「雪雰雰」」。
充「所沾」以「「翰」」以「「襟」」以「「裳」」以「「巾」」。
充「所贈」以「「金錯刀」」以「「琴琅玕」」以「「貂襜褕」」以「「錦繡段」」。
充「所報」以「「英瓊瑤」」以「「雙玉盤」」以「「明月珠」」以「「青玉案」」。
充「所感」以「「逍遙」」以「「惆悵」」以「「踟躕」」以「「增嘆」」。
充「所傷」以「「勞」」以「「傷」」以「「紆」」以「「惋」」。
```

```
凡「其所」中之「地」。
	夫「地」。書之。
云云。
吾有一言。曰「「此皆有所思之地也」」。書之。

凡「所贈」中之「寶」。
	夫「寶」。書之。
云云。
吾有一言。曰「「此皆美人之所贈也」」。書之。
```

```
有數一。名之曰「章」。
夫「其所」之長。為是其遍。
	吾有一言。曰「「我所思兮在」」。夫「其所」之「章」。書之。
	吾有一言。曰「「欲往從之」」。夫「所難」之「章」。書之。
	吾有一言。曰「「側身東望涕沾」」。夫「所沾」之「章」。書之。
	吾有一言。曰「「美人贈我」」。夫「所贈」之「章」。書之。
	吾有一言。曰「「何以報之」」。夫「所報」之「章」。書之。
	吾有一言。曰「「路遠莫致倚」」。夫「所感」之「章」。書之。
	吾有一言。曰「「何爲懷憂心煩」」。夫「所傷」之「章」。書之。
	加「章」以一。昔之「章」者。今其是矣。
云云。
```


```
我所思兮在太山
欲往從之樑父艱
側身東望涕沾翰
美人贈我金錯刀
何以報之英瓊瑤
路遠莫致倚逍遙
何爲懷憂心煩勞
我所思兮在桂林
欲往從之湘水深
側身東望涕沾襟
美人贈我琴琅玕
何以報之雙玉盤
路遠莫致倚惆悵
何爲懷憂心煩傷
我所思兮在漢陽
欲往從之隴阪長
側身東望涕沾裳
美人贈我貂襜褕
何以報之明月珠
路遠莫致倚踟躕
何爲懷憂心煩紆
我所思兮在雁門
欲往從之雪雰雰
側身東望涕沾巾
美人贈我錦繡段
何以報之青玉案
路遠莫致倚增嘆
何爲懷憂心煩惋
```

```
吾有二列。名之曰「复沓」曰「變辭」。
充「复沓」
以「「我所思兮在」」以「「欲往從之」」以「「側身東望涕沾」」
以「「美人贈我」」以「「何以報之」」
以「「路遠莫致倚」」以「「何爲懷憂心煩」」。
充「變辭」
以「其所」以「所難」以「所沾」以「所贈」以「所報」以「所感」以「所傷」。

有數一。名之曰「章」。
夫「其所」之長。為是其遍。
	有數一。名之曰「句」。
	夫「复沓」之長。為是其遍。
		夫「變辭」之「句」。夫其之「章」。名之曰「辭」。
    夫「复沓」之「句」。夫「辭」。書之。
    加「句」以一。昔之「句」者。今其是矣。
  云云。
	加「章」以一。昔之「章」者。今其是矣。
云云。

```
